# 带长期记忆的本地大模型聊天程序

一个基于Ollama的本地大模型聊天工具，支持**长期记忆**——程序会记住你们聊过的内容，下次启动时模型能回忆起之前说过的话。

## 功能

- 🧠 **长期记忆**：所有对话自动保存，下次启动时自动加载最近10条记忆
- 🤖 **本地大模型**：通过Ollama调用本地模型（支持Qwen、DeepSeek等），不需要API密钥，数据不外传
- 💾 **实时保存**：每轮对话都写入记忆文件，不怕丢失
- 🔧 **开箱即用**：自动安装依赖、自动创建记忆文件

## 如何运行

1. **安装Ollama**（如果没有）：https://ollama.com/
2. **下载模型**（以Qwen为例）：`ollama pull qwen:7b`
3. **确保Ollama服务运行中**：`ollama serve`
4. **修改代码中的模型名**：`MODEL_NAME = "qwen:7b"`（改成你下载的模型）
5. **运行程序**：`python memory_chat.py`

## 技术栈

- Python 3.8+
- Ollama（本地大模型部署框架）
- requests（API调用）
- 本地文件存储（长期记忆）

## 项目结构
memory_chat.py # 主程序
memory.txt # 记忆文件（自动生成）
